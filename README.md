# dbaptista.online – Portfólio Pessoal de D. Baptista
Este repositório contém o código e os recursos do site pessoal de D. Baptista, um portfólio online que apresenta habilidades técnicas, projetos, currículo e formulário de contato.
## Sobre
Olá! Eu sou D. Baptista, desenvolvedor backend com paixão por sistemas informáticos e interfaces minimalistas, intuitivas e fáceis de usar. Este site foi criado para apresentar minhas competências, projetos e experiências profissionais de forma clara e responsiva.
## Tecnologias e Habilidades
No site, são destacadas as habilidades técnicas que compõem meu repertório:
 - Java Desktop (Swing) – Criação de aplicações desktop com CRUD, integração com DAO e formulários otimizados.
 - Banco de Dados (MySQL) – Modelagem, normalização e otimização de banco de dados.
 - Design de Interface com Bootstrap – Interfaces web responsivas e modernas compatíveis com vários dispositivos.
 - Godot Engine – Desenvolvimento de jogos e aplicações iterativas com Godot Engine.
## Conteúdo do Site
O site tem as seguintes seções principais:
- Home
  Página inicial com uma saudação de boas-vindas e introdução pessoal.
- Projectos
  Lista e demonstração visual dos projetos desenvolvidos por você.
- CV
  Currículo profissional com formação, experiências e competências detalhadas.
- Fale Comigo
  Seção com formulário de contato para que visitantes do site possam enviar mensagens.
